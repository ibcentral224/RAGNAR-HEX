## 🚀 Installation

### Étape 1 — Fork / Upload sur GitHub

```bash
# Structure du dépôt
RAGNAR-HEX/
├── 📄 index.js       ← Serveur principal + QR Web
├── ⚡ commands.js    ← TOUTES les 40+ commandes
├── 📦 package.json   ← Dépendances
├── 🌐 render.yaml    ← Config Render.com
└── 🔒 .gitignore
```

1. Va sur **[github.com](https://github.com)**
2. Crée un nouveau dépôt → `RAGNAR-HEX`
3. Upload les 5 fichiers ci-dessus

---

### Étape 2 — Déployer sur Render.com

> **Gratuit · Sans serveur · Depuis ton téléphone Android**

```
1. 🌐 Ouvre render.com
2. 🔗 Connecte ton compte GitHub
3. ➕ Clique : New → Web Service
4. 📁 Sélectionne le dépôt : RAGNAR-HEX
5. ⚙️  Build Command  : npm install
6. ▶️  Start Command  : node index.js
7. 🚀 Clique : Deploy !
```

---

### Étape 3 — Scanner le QR Code

```
1. 📱 Ouvre l'URL fournie par Render
2. 📷 Un QR Code apparaît sur la page
3. 📲 WhatsApp → Appareils liés → Lier un appareil
4. ✅ Scanne → Bot connecté !
5. 🔔 Tu reçois une notification dans ton DM
```
